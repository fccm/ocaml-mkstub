
hxtr is a simple extractor for C header files (using gccxml).
  It prints the back the XML input from gccxml to different formatings,
  using templates.

The code is provided along the MIT license:
  https://secure.wikimedia.org/wikipedia/en/wiki/MIT_License

For more information read the local man page:
  $ man ./doc/man/hxtr.1
